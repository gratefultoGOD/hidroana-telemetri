# Vehicle Tracking System

A responsive vehicle tracking system with real-time data visualization using Leaflet maps and Chart.js.

## Features

- **Interactive Map**: Real-time vehicle tracking on OpenStreetMap using Leaflet
- **Multiple Vehicles**: Switch between different vehicles to track
- **Live Statistics**: Monitor current speed, distance traveled, and energy consumption
- **Data Visualization**: 
  - Speed over time line chart
  - Energy consumption bar chart
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- JavaScript (ES6+)
- Leaflet.js (for maps)
- Chart.js (for data visualization)

## Getting Started

1. Open `index.html` in a web browser
2. Select a vehicle from the dropdown menu
3. Watch the vehicle move on the map with real-time updates
4. Monitor speed and energy consumption charts

## Features Breakdown

- **Map Section**: Shows vehicle location with custom marker and route trail
- **Stats Cards**: Display current speed, distance, and energy consumption
- **Charts**: Visualize speed trends and energy usage over time
- **Vehicle Selector**: Switch between multiple tracked vehicles

The system updates every 2 seconds with simulated data for demonstration purposes.
